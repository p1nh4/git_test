Escreva um programa que pegue numa string e mostre o primeiro carácter 'a' que
que encontrar nela, seguido de uma nova linha. Se não houver caracteres 'a' na string
na cadeia, o programa escreve apenas uma nova linha. Se o número de parâmetros não for
1, o programa apresenta 'a' seguido de uma nova linha.

Exemplo:

$> ./aff_a “abc” | cat -e
a$
$> ./aff_a “dubO a POIL” | cat -e
a$
$> ./aff_a “zz sent le poney” | cat -e
$
$> ./aff_a | cat -e
a$
